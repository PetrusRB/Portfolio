# Meu portfolio

Servidor HTTP em C que roda html + typescript puro.

## Sobre

Esse projeto foi desenvolvido com intuito de eu aprender C e Redes.

## Tecnologias usadas

- **Frontend:** HTML5, CSS3, TypeScript
- **Build:** CMake/Build Script em Bash
